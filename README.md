# User Base
Sistema de gerenciamento de usuários via terminal, desenvolvido em Python com banco de dados SQLite.
---
## Sobre o projeto
O User Base permite cadastrar, buscar, editar, listar e excluir usuários diretamente pelo terminal, com validação de dados e armazenamento local em banco de dados SQLite. O banco de dados é gerado automaticamente na primeira execução do sistema, sem necessidade de configuração prévia.
---
## Funcionalidades
- Cadastrar usuário (nome, email, telefone)
- Buscar usuário por ID, email, nome ou telefone
- Listar todos os usuários cadastrados
- Editar email, nome ou telefone de um usuário
- Excluir usuário do sistema
